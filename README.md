# CR-ScorpionVenom-Nav
Code for statistical analysis from Akerman-Sanchez et al., 2026

In this repository, you can find the statistical analyses made by the authors of the cited paper. 
In the code, you may find "Var1" and "Var2" variables, which correspond to Activation and Inactivation curves, respectively. 
The multiple comparisons and diagnostic results can be found in the attatched pdf files.
